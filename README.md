
# Works SQLStream

**Works SQLStream** complements Apache Spark Structured Streaming with a set of hand-picked real-time
streaming data sources and destinations to build smart real-time data pipelines, especially for a wide
variety of Cyber Defense and IoT use cases.

## Sinks

### Aerospike

Aerospike is a database that enables the delivery of IoT-driven analysis and insights at lightning
speed to every key stakeholder. It can be used as a solid foundation for an intelligent real-time 
enterprise.

Works SQLStream for *Aerospike* complements Apache Spark Structured Streaming with an IoT-ready 
real-time database and makes it easy to build smart end-to-end data pipelines for wide variety of
use cases, from connected vehicles, health care and industrial manufacturing to smart buildings and
cities to transportation & logistics.
